# Security policy

Report vulnerabilities through GitHub's private
[security advisory form](https://github.com/pawnkit/pawnkit-cli/security/advisories/new).
Please do not open a public issue before a fix or mitigation is available.

Project files, lockfiles, adapter output, and subprocess output are untrusted.
Report credential leaks, unsafe process execution, path escapes, unbounded
resource use, or machine-readable output that exposes redacted values.
